# E-COMMERCE-ANALYTICS-DASHBORD

This project presents an interactive E-commerce Sales Dashboard built using Microsoft Excel.
It analyzes sales growth, profit trends, shipping cost, and country-wise performance.
Dynamic slicers are used to filter data by year, segment, and region.
Charts and pivot tables provide clear insights into product and category performance.
The dashboard supports data-driven decision-making through structured analysis and visualization
